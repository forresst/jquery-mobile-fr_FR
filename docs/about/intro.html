Aperçu de jQuery Mobile

La stratégie de jQuery Mobile peut être résumée simplement : un système d'interface utilisateur unifié qui fonctionne parfaitement sur toutes les plateformes des appareils mobiles les plus populaires et qui est construit sur les fondations solides de jQuery et jQuery UI. Axée sur des fonctionnalités riches, mais basé sur un code léger construit sur l'amélioration progressive avec un système de thème flexible et l'outil ThemeRoller.

Le framework inclut un système de navigation Ajax qui apporte les transitions de page animées et un ensemble de widgets UI : les pages, les dialogues, les barres d'outils, les listes, les boutons avec les icônes, les éléments de formulaire, les accordéons, les pliables et d'autres encore.

La différence essentielle avec notre approche est la grande variété de plateformes mobiles visées avec jQuery Mobile. Nous nous sommes focaliser pour que la fabrication de jQuery Mobile soit facile à apprendre avec un système simple basé sur le balisage pour appliquer le comportement et la thématisation. Pour des développeurs plus expérimentés, il y a une API riche d'options de configuration globale, des événements, des méthodes pour appliquer des scripts, générer des pages dynamiques et même créer des applications natives avec des outils comme PhoneGap.

Pour rendre ce support aussi large que possible, toutes les pages dans jQuery Mobile sont construites sur une base propre, la sémantique HTML, pour assurer la compatibilité avec quasiment n'importe quel appareil compatible Web. Parmi les dispositifs qui interprètent le CSS et le JavaScript, jQuery Mobile applique des techniques d'amélioration progressive pour transformer discrètement la page sémantique dans une expérience riche et interactive qui exploite la puissance de jQuery et CSS. Les fonctions d'accessibilité telles que WAI-ARIA sont étroitement intégrées dans le framework pour fournir un support pour les lecteurs d'écran et les autres technologies d'assistance.
