Accessibilité

jQuery Mobile est construit sur ​​le standard, la sémantique HTML, cela permet aux pages d'être accessibles sur la plus large gamme d'appareils possibles. Pour les navigateurs de grade A, de nombreux composants dans jQuery Mobile utilisent les techniques décrites dans la spécification WAI-ARIA du W3C, tels que la gestion du focus, la navigation au clavier et les attributs HTML.

En utilisant ces techniques, nous faisons de notre mieux pour assurer une expérience accessible aux utilisateurs ayant un handicap comme la cécité, qui peuvent utiliser les lecteurs d'écran (comme VoiceOver, le dispositif de l'iPhone d'Apple) ou d'autres technologies d'assistance pour accéder au web.
