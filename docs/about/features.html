Caractéristiques

Accueil Recherche

Principales caractéristiques :

  • Construit sur le noyau de jQuery afin d'avoir la syntaxe familière et cohérente de jQuery, de minimiser l'apprentissage et d'exploiter le code et les patterns de jQuery UI.
  • Compatible avec toutes les principales plateformes mobiles, tablettes, e-reader et de bureau - iOS, Android, Blackberry, Palm WebOS, Nokia/Symbian, Windows Phone 7, MeeGo, Opera Mobile/Mini, Firefox Mobile, Kindle, Nook et tous les navigateurs modernes avec différents niveaux de support.
  • Poids plume et un minimum de dépendance avec les images pour la performance
  • Architecture modulaire pour une création personnalisée qui est optimisée aux fonctionnalités uniquement nécessaires pour une application particulière.
  • Configuration axée sur le balisage HTML5 des pages et des comportements pour un développement rapide et un minimum de scripts requis.
  • Approche d'une amélioration progressive qui apporte le contenu et la fonctionnalité de base à tous les mobiles, tablettes et plateformes de bureau et une expérience semblable aux applications installées, comme sur les nouvelles plateformes mobiles.
  • Les techniques responsive design et les outils permettant un code source identique pour se mettre à l'échelle automatiquement par rapport à la taille des écrans sur un smartphone ou un écran de bureau.
  • Un puissant système de navigation basée sur Ajax pour permettre des transitions de page animée, tout en maintenant le bouton retour, les favoris et les URL propre à travers pushState.
  • Les fonctionnalités d'accessibilité telles que WAI-ARIA sont également incluses pour s'assurer que les pages fonctionnent pour les lecteurs d'écran (par exemple VoiceOver dans IOS) et d'autres technologies d'assistance.
  • Le support d'évènement tactile et de la souris rationalise les processus supportant le tactile, la souris et le curseur basé sur des méthodes de saisie utilisateurs à l'aide d'une simple API.
  • Des widgets UI unifiés pour des contrôles communs qui renforcent les contrôles natifs avec des contrôles tactiles optimisés, des contrôles thématisables qui sont agnostiques et faciles à utiliser.
  • Un puissant framework de thématisation et l'application Themeroller pour faire des expériences utilisateur très faciles à construire.
  • Pour avoir encore de meilleures performances, des paquets personnalisés peuvent être construit avec le Download builder
