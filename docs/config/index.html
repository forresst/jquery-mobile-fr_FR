Test de configuration globale

Les liens suivants causeront une actualisation complète de la page ainsi les options global de configuration prendront effets. Chaque lien ci-dessous réglera une option différente pour des tests rapides.

  • pushStateEnabled - False

    Amélioration pour utiliser history.replaceState dans les navigateurs pris en charge, pour convertir les URL basé sur le hachage Ajax dans le chemin du document complet.

  • touchOverflowEnabled - True

    Active le lissage des transitions de page et si c'est à vrai fixe les barres d'outils dans les périphériques qui supportent les propriétés CSS overflow: et overflow-scrolling: touch;.

  • defaultPageTransition - Fade

    Définit la transition par défaut pour les changements de page lors de l'utilisation d'Ajax. Définissez à 'none' pour qu'il n'y ait aucune transition par défaut.

  • defaultDialogTransition - Flip

    Définit la transition par défaut pour les changements de boîte de dialogue qui utilise Ajax. Définissez à 'none' pour qu'il n'y ait aucune transition par défaut.

  • minScrollBack - 999

    La distance minimale de défilement qui restera en mémoire lors du retour à une page.

  • loadingMessage - false

    Définit le texte qui apparaît quand une page se charge. Si c'est à false, le message n'apparaîtra pas du tout.

  • pageLoadErrorMessage - "Oups, nous avons cassé Internet !"

    Définit le texte qui apparaît quand une page ne se charge pas grâce à l'Ajax.

  • meta tag apple-mobile-web-app-capable

    Voir les docs sans navigateur chrome dans iOS.

  • jQuery core version 1.7 Beta 1

    Teste les docs avec la dernière version de jQuery
