Mise à l'échelle d'images: exemples de Lightbox

Le framework de CSS contient des règles pour rendre à l'échelle les images qui sont les enfants immédiats de la popup pour s'adapter à l'écran. En raison de la position absolue du conteneur de la popup et de l'écran, la hauteur n'est pas ajustée sur la hauteur de l'écran sur tous les navigateurs. Vous pouvez empêcher le défilement vertical avec un simple script qui définit le max-height de l'image.

Dans les deux exemples ci-dessous les divs avec data-role="popup" ont une classe de photopopup.

Le gestionnaire est lié à l'événement popupbeforeposition qui assure que l'image est non seulement mise à l'échelle avant qu'elle ne soit affichée, mais aussi lorsque la fenêtre est redimensionnée (changement d'orientation par exemple). Dans cet exemple de code de la hauteur est réduite de 60 pour prendre en compte une tolérance du haut et du bas de 30 pixels.


$( document ).on( "pageinit", function() {
    $( ".photopopup" ).on({
        popupbeforeposition: function() {
            var maxHeight = $( window ).height() - 60 + "px";
            $( ".photopopup img" ).css( "max-height", maxHeight );
        }
    });
});
