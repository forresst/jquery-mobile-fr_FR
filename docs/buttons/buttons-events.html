Évènements liés directement à l'élément a, input ou button. Utilisez les événements virtuels de jQuery Mobile ou liez les événements standard de JavaScript, comme change, focus, blur, etc.. :

 
$( ".myButton" ).bind( "click", function(event, ui) {
  ...
});

Le plugin bouton de formulaire a les événements personnalisés suivants :

create déclenché quand un bouton de formulaire est créé.

$('[type='submit']').button({
   create: function(event, ui) { ... }
});
