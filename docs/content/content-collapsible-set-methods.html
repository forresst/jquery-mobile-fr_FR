Le plugin ensemble de pliable a les méthodes suivantes :

refresh mise à jour de l'ensemble de pliable

Si vous manipulez un ensemble de pliable via JavaScript (par exemple vous ajoutez des conteneurs de pliable), vous devez appeler la méthode refresh sur elle même pour mettre à jour le style visuel.


$('.selector').collapsibleset('refresh');
