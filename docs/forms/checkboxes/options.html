La case à cocher a les options suivantes :

mini boolean

par défaut : false

Définit la taille de l'élément pour qu'il soit plus compact : une version mini. Cette option est également définie comme un attribut de données : data-mini="true"

$("input[type='checkbox']").checkboxradio({ mini: "true" });
theme string

par défaut : null, hérité d'un parent

Définit le jeu de couleurs (échantillon) pour toutes les instances de ce widget. Il accepte une seule lettre de A à Z qui correspond aux échantillons inclus dans votre thème. Par défaut, il hérite de la même couleur de l'échantillon que possède le conteneur parent s'il ne est pas explicitement défini. Cette option est également définie comme un attribut de données : data-theme="a"

$("input[type='checkbox']").checkboxradio({ theme: "a" });
