La case à cocher a les méthodes suivantes :

enable active une case à cocher désactivée

 $("input[type='checkbox']").checkboxradio('enable');
disable désactive une sélection

$("input[type='checkbox']").checkboxradio('disable');
refresh met à jour une sélection personnalisée
Si vous manipulez une case à cocher via JavaScript, vous devez appeler la méthode refresh sur elle même pour mettre à jour le style visuel. 

$("input[type='checkbox']:first").attr("checked",true).checkboxradio("refresh");
