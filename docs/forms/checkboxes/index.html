Les cases à cocher sont utilisées pour fournir une liste d'options dont plusieurs peuvent être sélectionnés. Les cases à cocher traditionnelles de bureau ne sont pas optimisés pour la saisie tactile donc dans jQuery Mobile, nous stylisons le label pour les cases à cocher de sorte qu'elles soient plus grandes et cliquables visuellement. Un ensemble personnalisé d'icônes sont ajoutées au label pour fournir d'autres informations visuelles.

Les contrôles du radio bouton et de la case à cocher ci-dessous utilisent le balisage standard input/label, mais sont stylisés pour être plus "tactiles". Le contrôle stylisé que vous voyez est réellement l'élément label, qui se trouve sur l'input réel, donc si les images ne se chargent pas, vous aurez encore un contrôle fonctionnel. Dans la plupart des navigateurs, en cliquant le label cela déclenche automatiquement un clic sur l'input, mais nous avons du déclencher la mise à jour manuellement pour quelques navigateurs mobiles qui ne font pas cela nativement. Sur le bureau, ces contrôles sont accessible par le clavier et le lecteur d'écran. Regardez la reference de data- attribute pour voir tous les attributs possibles que vous pouvez ajouter aux cases à cocher.

Pour créer une seule case, ajoutez un input avec un attribut type="checkbox" et un label correspondant. Si le input n'est pas enveloppé dans le label correspondant, veuillez définir l'attribut for du label pour correspondre à l'ID de input ainsi ils seront sémantiquement associés.

	
<label><input type="checkbox" name="checkbox-1" /> J'accepte </label>
			
<input type="checkbox" name="checkbox-0" id="checkbox-0" class="custom" />
<label for="checkbox-0">J'accepte</label>

Ceci va produire deux cases à cocher de base. Les styles par défaut auront la largeur de l'élément définies à 100% du conteneur parent.

Version mini

Pour avoir une version plus compacte, qui est utile dans les barres d'outils et les espaces restreints, ajoutez l'attribut data-mini="true" à l'élément pour créer une version mini. 

	
<input type="checkbox" name="checkbox-0" id="checkbox-mini-0" class="custom" data-mini="true" />
<label for="checkbox-mini-0">J'accepte</label>

Ceci va produire une sélection pas aussi grande que la version standard et disposant d'une taille inférieure.

Conteneurs de champs et de légendes

Parce que les cases à cocher utilisent l'élément label pour le texte affiché à côté de l'élément de formulaire de la case à cocher, nous recommandons l'enveloppement de la case à cocher dans un élément fieldset qui a une legend qui agit comme le titre de la question. Ajouter l'attribut data-role="controlgroup" au fieldset de sorte qu'il puisse être stylisé de manière parallèle comme les text inputs, selects ou autres éléments de formulaire.

Enveloppez le fieldset dans un div avec l'attribut data-role="fieldcontain" de sorte qu'il puisse être stylisé de manière parallèle comme les text inputs, selects ou autres éléments de formulaire.

	
<div data-role="fieldcontain">
    <fieldset data-role="controlgroup">
	   <legend>Acceptez les termes :</legend>
	   <input type="checkbox" name="checkbox-1" id="checkbox-1" class="custom" />
	   <label for="checkbox-1">J'accepte</label>
    </fieldset>
</div>

Cases à cocher regroupées verticalement

Typiquement, il y a plusieurs cases à cocher figurant sous un titre de la question. Pour intégrer visuellement de multiples cases à cocher dans un ensemble groupé de bouton, le framework va automatiquement supprimer toutes les marges entre les boutons et arrondir seulement les coins supérieurs et inférieurs de l'ensemble s'il y a un attribut data-role="controlgroup" sur le fieldset.

Ensembles horizontal à bascule

Les cases à cocher peuvent aussi être utilisé pour un ensemble de boutons groupés où plus d'un bouton peut être sélectionné à la fois, comme le groupe de boutons gras, italique et souligné vu dans un traitement de texte. Pour faire un jeu de bouton horizontale, ajoutez data-type="horizontal" à fieldset.

<fieldset data-role="controlgroup" data-type="horizontal">

Le framework va positionner les labels de sorte qu'ils soient positionnés côte à côte sur une ligne, masquer les icônes des cases à cocher et arrondir seulement sur les bords gauche et droit du groupe.
