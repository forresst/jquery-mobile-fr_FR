Le plugin menu de sélection a les méthodes suivantes :

close Fermer un menu de sélection ouvert

$('select').selectmenu('close');
enable active une sélection désactivée

$('select').selectmenu('enable');
disable désactive une sélection

$('select').selectmenu('disable');
open Ouvrir un menu de sélection fermé

$('select').selectmenu('open');
refresh met à jour une sélection personnalisée
Ceci est utilisé pour mettre à jour la sélection personnalisée pour refléter la valeur de l'élément d'une sélection native. Si le nombre d'options dans la sélection est différent du nombre d'éléments dans le menu personnalisé, il reconstruira le menu personnalisé. Aussi, si vous passez un argument à true, vous pouvez forcer que la reconstruction se produise. 

// actualisation des valeurs
$('select').selectmenu('refresh');

// actualisation et forcer la reconstruction
$('select').selectmenu('refresh', true);
