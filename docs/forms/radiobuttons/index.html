Les radio boutons sont utilisées pour fournir une liste d'options dont un seul élément peut être sélectionné. Les radio boutons traditionnels de bureau ne sont pas optimisés pour la saisie tactile donc dans jQuery Mobile, nous stylisons le label pour les radio boutons de sorte qu'ils soient plus grands et cliquables visuellement. Un ensemble personnalisé d'icônes sont ajoutées au label pour fournir d'autres informations visuelles.

Les contrôles du radio bouton et de la case à cocher ci-dessous utilisent le balisage standard input/label, mais sont stylisés pour être plus "tactiles". Le contrôle stylisé que vous voyez est réellement l'élément label, qui se trouve sur l'input réel, donc si les images ne se chargent pas, vous aurez encore un contrôle fonctionnel. Dans la plupart des navigateurs, en cliquant le label cela déclenche automatiquement un clic sur l'input, mais nous avons du déclencher la mise à jour manuellement pour quelques navigateurs mobiles qui ne font pas cela nativement. Sur le bureau, ces contrôles sont accessible par le clavier et le lecteur d'écran. Regardez la reference de data- attribute pour voir tous les attributs possibles que vous pouvez ajouter aux radio boutons.

Radio boutons regroupés verticalement

Pour créer un ensemble de radio boutons, ajoutez un input avec l'attribut type="radio" et un label correspondant. Réglez l'attribut for de label pour faire correspondre l'ID de l'input ainsi ils sont sémantiquement liés.

Parce que les radio boutons utilisent l'élément label pour le texte affiché à côté de l'élément de formulaire du radio bouton, nous recommandons l'enveloppement du radio bouton dans un élément fieldset qui a une legend qui agit comme le titre de la question.

Pour intégrer visuellement de multiples radio boutons dans un ensemble groupé de bouton, le framework va automatiquement supprimer toutes les marges entre les boutons et arrondir seulement les coins supérieurs et inférieurs de l'ensemble s'il y a un attribut data-role="controlgroup" sur le conteneur.

	
<fieldset data-role="controlgroup">
	<legend>Choisissez un animal :</legend>
     	<input type="radio" name="radio-choice-1" id="radio-choice-1" value="choice-1" checked="checked" />
     	<label for="radio-choice-1">Chat</label>

     	<input type="radio" name="radio-choice-1" id="radio-choice-2" value="choice-2"  />
     	<label for="radio-choice-2">Chien</label>

     	<input type="radio" name="radio-choice-1" id="radio-choice-3" value="choice-3"  />
     	<label for="radio-choice-3">Hamster</label>

     	<input type="radio" name="radio-choice-1" id="radio-choice-4" value="choice-4"  />
     	<label for="radio-choice-4">Lézard</label>
</fieldset>

Ceci va produire un ensemble de radio bouton groupé verticalement. Les styles par défaut définissent la largeur du groupe de boutons à 100% du conteneur parent et empile les labels sur une ligne distincte.

Version mini

Pour avoir une version plus compacte, qui est utile dans les barres d'outils et les espaces restreints, ajoutez l'attribut data-mini="true" à l'élément pour créer une version mini. 

			
<fieldset data-role="controlgroup" data-mini="true">

    	<input type="radio" name="radio-choice-1" id="radio-mini-1" value="choice-1" checked="checked" />

    	<label for="radio-mini-1">Crédit</label>
    	<input type="radio" name="radio-choice-1" id="radio-mini-2" value="choice-2"  />

    	<label for="radio-mini-2">Débit</label>
    	<input type="radio" name="radio-choice-1" id="radio-mini-3" value="choice-3"  />

    	<label for="radio-mini-3">Encaisser</label>
</fieldset>

Ceci va produire un radio bouton pas aussi grand que la version standard et disposant d'une taille inférieure.

Conteneurs de champs

Optionnellement, enveloppez les radios bouton dans un conteneur avec l'attribut data-role="fieldcontain" afin de les grouper visuellement dans un formulaire plus long.

	
<div data-role="fieldcontain">
    <fieldset data-role="controlgroup">
    	<legend>Choisissez un animal :</legend>
         	<input type="radio" name="radio-choice-1" id="radio-choice-1" value="choice-1" checked="checked" />
         	<label for="radio-choice-1">Chat</label>

         	<input type="radio" name="radio-choice-1" id="radio-choice-2" value="choice-2"  />
         	<label for="radio-choice-2">Chien</label>

         	<input type="radio" name="radio-choice-1" id="radio-choice-3" value="choice-3"  />

         	<input type="radio" name="radio-choice-1" id="radio-choice-4" value="choice-4"  />
         	<label for="radio-choice-4">Lézard</label>
    </fieldset>
</div>

Ensemble horizontal de radio boutons

Les radio boutons peuvent également être utilisés pour un ensemble de bouton groupé défini dont uniquement un seul bouton peut être sélectionné à la fois, comme un contrôle visuel d'un interrupteur. Pour faire un jeu de radio bouton horizontale, ajoutez data-type="horizontal" à fieldset.

<fieldset data-role="controlgroup" data-type="horizontal" >
Mise en page de la vue :

Le framework va positionner les labels de sorte qu'ils soient positionnés côte à côte sur une ligne, masquer les icônes des radio boutons et arrondir seulement sur les bords gauche et droit du groupe.
