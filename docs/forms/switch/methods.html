Le plugin curseur a les méthodes suivantes :

enable active un curseur désactivé

$('.selector').slider('enable');
disable désactive un curseur

$('.selector').slider('disable');
refresh met à jour un curseur

Si vous manipulez un curseur via JavaScript, vous devez appeler la méthode refresh sur elle même pour mettre à jour le style visuel.

			
$('.selector').slider('refresh');
