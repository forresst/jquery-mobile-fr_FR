Le mode reflow a une option, classes, qui n'est configurable que via JavaScript car il attend une valeur littérale d'objet. L'option classes a deux propriétés qui définissent les noms de classes structurelles que le plugin utilise :

classes.reflowTable (par défaut : "ui-table-reflow")
Classe ajoutée au contenu de l'étiquette générée, ajouté à chaque cellule de la table sur la base du nom de l'entête
classes.cellLabels (par défaut : "ui-table-cell-label")
Classe ajoutée à la première cellule à l'intérieur de chaque entête groupée de colonne. Cela rend plus facile pour styliser différemment le visuel qui délimite les groupes de colonnes.
