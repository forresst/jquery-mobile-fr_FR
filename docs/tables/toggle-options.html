La table columntoggle possède plusieurs options :

columnBtnTheme par défaut : null
Définit le thème pour le bouton de sélection de colonne. Définissez-le avec n'importe quelle lettre de nuancier valide dans votre thème. Disponible sous l'attribut data-column-btn-theme.
columnBtnText par défaut : "Columns..."
Définit le texte pour le bouton de sélection de colonne. Disponible sous l'attribut data-column-btn-text.
columnPopupTheme par défaut : null
Définit le thème pour les cases à cocher du sélecteur de colonne. Définissez-le avec n'importe quelle lettre de nuancier valide dans votre thème. Disponible sous l'attribut data-popup-theme.

Le mode columntoggle a aussi une option, classes, qui n'est configurable que via JavaScript car il attend une valeur littérale d'objet. L'option classes a deux propriétés qui définissent les noms de classes structurelles que le plugin utilise :

classes.columnToggleTable par défaut : "ui-table-columntoggle"
Classe assignée à la table.
classes.columnBtn par défaut : "ui-table-columntoggle-btn"
Classe assignée au bouton de sélecteur de colonne.
classes.popup par défaut : "ui-table-columntoggle-popup"
Classe assignée à la popup de sélection de colonne.
classes.priorityPrefix par défaut : "ui-table-priority-"
Préfixe de classe ajoutée à chaque cellule dans une colonne. Cette chaîne est ajoutée à la valeur de priorité mise sur les entêtes.
